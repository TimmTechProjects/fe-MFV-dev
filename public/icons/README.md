# Icons & App Images

This directory contains app icons for various platforms and purposes.

## Required Icons

### Favicons
- **favicon.ico** (32x32px, ICO format)
  - Browser tab icon
  - Should be clear at small size

- **icon.png** (32x32px, PNG)
  - Modern browsers
  - Alternative to ICO

- **apple-icon.png** (180x180px, PNG)
  - iOS home screen
  - iPad
  - High resolution

### PWA Icons
- **icon-192.png** (192x192px, PNG)
  - Android home screen
  - Chrome app install

- **icon-512.png** (512x512px, PNG)
  - Android splash screen
  - High-quality app icon

### Shortcut Icons (Optional, for PWA shortcuts)
- **marketplace.png** (96x96px)
- **vault.png** (96x96px)
- **forum.png** (96x96px)

## Design Guidelines

### Color Scheme
- Primary: #22c55e (Green)
- Background: #0a0a0a (Dark mode) / #ffffff (Light mode)

### Icon Style
- Simple, recognizable at small sizes
- Use brand colors
- Consider both light and dark backgrounds
- Test at various sizes

### Export Settings
- Format: PNG with transparency
- Color space: sRGB
- Resolution: 72 DPI minimum

## Tools

- [Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [Figma](https://figma.com) or [Canva](https://canva.com) for design

## Testing

- Test favicon in multiple browsers
- Test PWA icons on actual devices
- Check both light and dark modes
- Verify transparency works correctly
